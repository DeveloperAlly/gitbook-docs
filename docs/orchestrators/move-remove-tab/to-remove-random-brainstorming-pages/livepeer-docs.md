# Livepeer Docs

### Livepeer Overview

Livepeer is building the world's

Open source media stack & server

* combo of compute infrastructure
* and video software for building applications

Deliver video globally

Integrates with CDNs

Integrates with Storage

Social Media (lens & farcaster)

Power Event & Live Streaming

Power twitch videos.

FredAgain, Skrillex Stream on Livepeer (LatRadio - streams from a shipping container in Brooklyn 24/7)
